# Pelumi's Playground

A personal project showcase smy list of AI tools i've built.

View here:

- **HTTPS:** https://p31umi.github.io/Playground/



## Adding or editing projects

Edit the **`PROJECTS`** array in **`index.html`** (inline `<script>` block above `main.js`). Each item is `{ title: '...', description: '...', url: 'https://...' }`. Save and refresh the site.

## Tech

- HTML, CSS (Flexbox/Grid), Vanilla JavaScript
- Roboto Mono font (Google Fonts)
- Lucide icons (CDN)
